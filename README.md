# Stranger Things

This is a website made to pay homage to Stranger Things.
The website offers you a look at all the **main characters** and **creatures** that can be found in the **Upside Down**.
It also shows you all the episodes in the show - their names and descriptions.

## Set-up

1. Run `npm i` at the root of the project to install all the dependencies.
2. Run `npm run dev` to start the server and see the result.
